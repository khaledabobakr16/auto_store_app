# Auto_Store_App

Auto store application using Firebase

## What Firebase features are implemented?

1-Firebase Authentication
  - Email based sign up/in.
  - Google Sign in & Facebook Sign in.
  - Rest Password.
    
2-Cloud Firestore

3-Firebase Storage
